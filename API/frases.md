```
Él bebo agua -> Él agua beber
```
```
El niño bebe agua -> niño agua beber
```
```
La niña bebe agua con limón -> niña agua limón con beber
```
```
Mi tía fue al supermercado en coche. -> yo tía ayer supermercado coche
```

```

```
```

```
